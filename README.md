                                            Проектная работа "Научиться учиться". 
Проект представляет собой информацинную страничку для студентов или людей, которые обучаются новому. Данный веб-сайт одностраничный, содержит ссылки на фишки для качественного усвоения информации. 
При создании проекта была использована методология БЭМ и схема Nested организации файловой структуры. Для вёрстки страницы использована технология CSS Flexbox, настроена кроссбраузерность для flexbox. На сайте присутствует анимация, контентные картинки, настроено различное позиционирование элементов.
В дальнейшем планируется адаптировать сайт для мобильных и планшетов.
