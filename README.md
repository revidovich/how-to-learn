                                            Проектная работа "Научиться учиться". 
Сайт представляет собой информацинную страничку для студентов или людей, кторые обучаются новому. Содержит ссылки на фишки для качественного уcвоения информации. 
При создании проекта была использована методология БЭМ и схема Nested организации файловой структуры. Вёрстка страницы - с помощью технологии CSS Flexbox.
